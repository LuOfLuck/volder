Â¿como crear un Proyecto y como funciona?

Explicacion breve:

Para crear un proyecto debe de indicar el titulo, una breve descripcion de que se tratara y al curso que esta dirigida.
Al crear un proyecto podra entrar a el y crear grupos de alumnos donde seleccionara el los estudiantes que quiere que este en ese grupo (evidentemente cada grupo de alumno solo servira para el proyecto al que este incluido).
Los grupos de alumno no tienen limite de estudiantes, sinembargo solo se podran agregar estudiantes del curso que selecciono posteriormente.
Cuando cree un grupo de alumnos podra dar 1 o varias actividades, cada actividad contara con un chat exclusivo para ese actividad del grupo.
Aun asi cada proyecto tendra un chat general.
